AI Berkshire — 巴菲特、芒格、段永平、李录在 Claude Code 里吵架帮你买股票

xbtlin 用 Claude Code 做了一套价值投资研究框架，不是让 AI 分析财报那种老套路。它把巴菲特、芒格、段永平、李录四位大师的方法论塞进四个 Agent，让它们各自独立研究同一家公司，然后互相吵架。巴菲特说拼多多估值极低可以买（4.4/5），李录说管理层文化有隐患十年确定性不足（2.0/5），这种冲突输出才是真参考。

我看完的实话说
说实话这比直接问 AI"帮我分析一下拼多多"有意思。四个大师视角互相吵架，巴菲特说买、李录说不买——不是那种"一方面另一方面"的废话，是真正帮你做判断的冲突输出。实盘数据是跑出来的。最让我在意的是快速否决清单：管理层有诚信问题直接否决，不管估值多低。

Voicebox — Spacedrive 作者做了个免费本地 ElevenLabs

Jamie Pine 是 Spacedrive（开源文件管理器）的作者，这次做了 Voicebox——本地优先的 AI 语音工作室，免费开源。几秒音频克隆声音，7 个 TTS 引擎，全局热键听写直接粘贴到任何应用，内置 MCP 服务让 Claude Code 和 Cursor 调用你的语音能力。所有模型本地跑，数据不上传。

我看完的实话说

独立 AI Builder 真实数据 — 90% 失败

youngju.dev 个人博客分析了 9 个独立 AI Builder 的真实案例——Cursor、Perplexity、Lovable、Vercel v0。结论很直白：90% 失败。但失败的案例比成功的更有参考价值——你看到别人在哪里摔了，比看别人在哪里赢了更有用。还分析了 2026 年一人公司的标准技术栈：Cursor/Claude Code + Bun/Hono + Cloudflare/Vercel + Stripe。

我看完的实话说
90% 失败不意外。但看到具体是谁失败了、怎么失败的，才是有用的部分。youngju 不是告诉你"学他们就能成功"，而是"他们赌的方向和时机你未必有"。最实用的是那个 2026 标准技术栈——不是高大上的架构，就是 Cursor + Bun + Cloudflare 几样拼在一起，一个人就能跑。
